다음과 같은 특징을 가지고 있습니다

공공 데이터 사이트를

지상, 해양, 고층, 항공관측, 위성, 레이더, 수치예보모델자료 등 총 30종류의 날씨데이터를 다운로드 받을 수 있습니다.

원하는 특성과 값으로 데이터 조회가 가능합니다

MySQL과 PHP로 DataBase의 값을 조회 할수 있습니다.

Python 파일을 그대로 사용할수 있습니다.

별도 수정없이 .py파일을 실행한 localhost 서버에 접속 가능합니다.

공모전에서 제공하는 데이터로 나만의 빅데이터 플랫폼을 만들어보세요.

원하는 빅데이터 플랫폼을 제작하고 이를 활용하여 예측모델을 생성할수 있습니다/

개발자를 위한 오픈 API 서비스를 이용해보세요

Open API로 데이터를 실시간 업데이트 할수 있습니다.

이렇게 많은 데이터 어디에 쓰이나요?

  • 날씨가 상품 매출에 미치는 영향을 분석할 때
  • 인체에 날씨가 미치는 영향을 연구할 때
  • 날씨에 따른 작물 성장 상태를 파학하고 싶을 때
  • 작물의 파종·수확 시기를 준비해야할 때
  • 에너지 사업을 계획할 때
  • 건설 입지를 선정할 때
